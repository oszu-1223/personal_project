# 네트워크 취약점 탐색 프로그램


## 프로젝트 소개
* 네트워크 서버의 취약점을 탐색하는 과정을 게임 형태로 구현한 프로그램
* 사용자가 웹 서버, 로그인 서버, 데이터베이스 서버 중 하나를 선택하여 각 서버의 보안 항목을 검사하고, 발견된 취약점에 따라 위험 점수를 산정함
